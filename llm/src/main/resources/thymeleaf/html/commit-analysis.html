How Files Scope & Density Work

Touching more files means broader impact — more review surface, more integration risk. The scope multiplier rewards breadth, but a density dampener prevents shallow changes (renames, import-only edits) from inflating the score.

Scope Multiplier

1.0 + files0.75 × 0.40 × density

8 files2.90×
25 files5.47×
50 files8.50×

Values shown at full density (1.0).

Density Dampener

min(1.0, avgLinesPerFile / 10)

≥10 lines/file1.00

2 lines/file0.20
1 line/file0.10

Shallow edits get reduced scope credit.

Example: 150 lines across 50 files (avg 3 lines/file) → density = 0.30, multiplier = 1.0 + 7.51 × 0.30 = 3.25×. Same 150 lines across 8 files (avg 19 lines/file) → density = 1.00, multiplier = 2.90×. The focused change scores similarly despite fewer files because density is full.
