## What This Report Shows

We scan the **entire project** for duplicated code blocks, then filter to show only those that **overlap with lines you changed**.
A duplicate is "introduced" if your changes created or significantly modified the duplicated code.

### Quick Summary

| What | Count | What It Means |
|------|-------|---------------|
| **Duplicated Blocks** | [(${totalClones})] | Code blocks that appear more than once and touch your changes |
| Duplicated Lines | [(${totalDuplicatedLines})] | Total lines of duplicated code |
| Duplication Rate | [(${#numbers.formatDecimal(duplicatedPercentage, 1, 1)})]% | Percentage of changed code that is duplicated |
